## ' <= '   小于等于号；若判断正确，输出' true ',若判断错误，输出' false '

## ' >= '   大于等于号；若判断正确，输出' true ',若判断错误，输出' false '

## ' == '   检查左右两边的值是否相等; 若判断正确，输出' true ',若判断错误，输出' false '

## ' 运算优先级(PEMDAS) '    括号(Parentheses)，指数(Exponents)，乘(Multiply)，除(Division)，加(Adding)，减(Subtraction).  这个是Python的运算优先级.  同时，优先级是，括号和指数是同一级，乘法和除法是同一级，而加法和减法是同一级。  相当于 " P E( M & D )( A & S ) "

## ' variable(变量) '    用来代指其他东西的名字，同时可以通过变量名来了解这个变量的作用; 变量名由字母，数字，下划线(_),同时变量名的开头只能用字母或者下划线开头，不能由数字开头

## ' = '    赋值，让变量得到(get)一个值(value)

## ' _ '    下划线，无实际作用，但建议在写变量时使用；同时不要用空格来命名变量和文件名

## ' 整数(int) '    数据是整数形式

## ' int() '    将数据变为整数

## ' 浮点数(float) '     数据有小数

## ' float() '    将数据变为浮点数

## ' {}（花括号） '   把变量嵌入，并使用的符号(前提是在字符串前面还有字符' f '(format),表示将花括号内的变量打印出来)

## ' f(format, 格式) '    写在字符串前面，将字符串中的变量引入（f-string）

## ' .format() '    格式化字符，比如 ' str.format("False", "Ture") ',然后我可以在另一个变量里面调用这个类似数组的排列形式：y = " {1},{0}" ，用{}花括号来调用()括号中的字符，第一个为'0'；若在{}花括号中为空，即按数组顺序依次输出。

## ' round() '    对浮点数进行四舍五入处理；第一个参数是浮点数本身，第二个参数是小数点后保留的位数;如果不提供第二个参数或第二个参数为0,则四舍五入到最近的整数

## ' 关键字 '      python自带的关键字，他们有python内部设置的功能，如果加了引号，就变成字符串。比如: Ture, False

## ' \n '   切换到下一行。反斜杠的名字叫：转义符号。用反斜杠加上对应字母，可以实现许多功能，比如：' \t '是一个制表符; ' \\ '输出反斜杠
### ' \' ':单引号; ' \" ':双引号；' \a '：ASCII响铃符(BEL); ' \b '：ASCII退格符(BS); 
### ' \f '：ASCII进纸符(FF); ' \n '：ASCII换行符(LF); ' \N{name} ': Unicode数据库中的字符名，其中name是它的名字，仅Unicode适用； ' \r '：ASCII回车符(CR); ' \t '：ASCII水平制表(TAB);  ' \uxxxx ':值为16位十六进制值xxxx的字符； ' \Uxxxxxxxx ':值为32位十六进制值xxxxxxxx的字符 ；  ' \v ':ASCII垂直制表符(VT); ' \ooo ':值为八进制值ooo的字符；' \xhh ':值为十六进制值hh的字符；

## ' int(input()) '     将得到的字符串转换为整数

## ' import '   导入对应的模块(module)(相当于函数库)，比如：" import math ": 导入math模块; " import sys "：导入sys模块

## ' from import '  从对应的模块(相当于函数库)，导入需要的函数。比如：" from sys import argv "：从sys模块导入对应的argv函数

## ' argv '    处理命令行参数，存储命令行调用python脚本时提供的参数.(sys.argv) <--真实路径.   argv是在" 执行命令时 "就输入参数.格式：" scripts, time, weight = argv ",说明有三个变量，变量scripts输出：脚本名字； 变量time输出：对应的值；同时你运行脚本时，需要输入正确数量的值，同时scripts不需要接收输入，所以执行时应该用两个参数：" test.py time weight "

## ' input() '   从标准输入读取一行文本,允许从用户那里接收输入.如果想将其他变量得到输出，可以在括号中调整变量值.或者输入提示符.  input是在" 脚本运行过程 "中的到参数.

## ' 参数变量 '     保持对应的参数.在运行完当前脚本时，此变量将按程序变化，同时输出。

## ' 解包 '  相当于将值赋值给对应的变量

##  ' open() '    内置函数，打开一个文件，创建一个file对象.他不是返回文件的内容，而是返回 '文件对象'.例如：open(filename)。模式：w（写入）， w+，r+，a(追加)  open(foo, 'w'),w模式为如果文件不存在，python自动创建；如果文件存在，python打开他并清空内容.

## ' .truncate() '    清空文件。（小心使用）

## ' .write('stuff') '    将' stuff '写入文件。 write接收一个字符串作为参数，并将字符串写入文件。

## ' exists '    将文件名字符串作为参数，如果存在，那就返回True；如果不存在，那就返回False。命令导入：from os.path import exists
